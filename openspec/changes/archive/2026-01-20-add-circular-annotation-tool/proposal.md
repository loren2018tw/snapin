# Change: Add Circular Annotation Tool

## Why

To enhance the drawing app with a circular annotation tool, allowing users to draw circles for annotations and highlighting areas.

## What Changes

- Add 'Circle' tool to the drawing functionality in drawing.html
- Implement circle drawing logic similar to rectangle tool
- Add circle shapes to the shapes array for persistence and redrawing
- Reuse existing rectangle color for circle strokes

## Impact

- Affected specs: drawing
- Affected code: public/drawing.html
- No breaking changes
